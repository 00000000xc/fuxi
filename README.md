## Getting Started

Dependency: `Linux` `python3.x` `redis` `mongoDB`

Get the project:

```shell
git clone https://github.com/jeffzh3ng/fuxi.git
cd fuxi
pip install -r requirements.txt
```

Creating configuration file:

```shell
cd instance/
mv _config.py config.py
vi config.py
```

Begin using `fuxi`

```shell
chmod +x fuxi_manage.sh
./fuxi_manage.sh
```
